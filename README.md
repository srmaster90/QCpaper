Code to generate figures from Unsihuay D and Master SR, "Extending Quality Control Procedures for Multiplexed Clinical Assays", *Clinical Chemistry*, 2026.
Note that generation of QC power curves requires that the user first install the *qcsim* package, found at **https://github.com/srmaster90/qcsim**.
Instructions for installing the *qcsim* package are found in the README for that repository.  Additionally, documentation of the *qcsim()* function is found within help files provided with the *qcsim* package.

*qcsim_simdata.R* calculates QC power curves using the *qcsim* library.  Data are saved in an .RData file.  For convenience, given the time required to calculate results for highly multiplex simulations, the resulting .Rdata file is also provided in the Results directory.

*Plot_Figs.R* uses the .Rdata files generated by *qcsim_simdata.R* in order to produce the figures shown in the paper.

*PerturbQC.R* simulates and plots results from perturbation of a multiplex pattern assessed with several machine learning classifiers.
